# 解析手法  
xmlに記述されているbase64のコードを2進数に変換し、同じ2進数の繰り返し部分を探しながら、
パラメータ箇所を探っていく。  
base64をそのままテキストに変換しない理由は単純で簡単に復号化出来ないから。
復号対策なのか不要で無意味な0が大量に挿入されているので、2進数で探っていく必要がある。

# 手順
1. xmlからbase64データ取得
2. 改行全て解消
3. pythonコードで2進数に変換
4. 同じ繰り返しを探していく
